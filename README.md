# react-dashboard
Bu layihə interaktiv bir Dashboard-dur.

### Özəlliklər:
* **Drag & Drop:** Kartların yerini sürüşdürərək dəyişmək olar.
* **Mock API:** Məlumatlar JSONPlaceholder-dən gəlir.
* **Responsive:** Mobil, planşet və masaüstü cihazlara uyğundur.

### Necə işlətməli?
1. Layihəni kopyalayın.
2. `npm install` yazaraq kitabxanaları yükləyin.
3. `npm start` yazaraq layihəni başladın.
